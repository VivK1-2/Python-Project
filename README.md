Expense Tracker Chatbot

An interactive Expense Tracker Bot made with Python and PyScript, allowing users to record and categorize their expenses directly from their browser. This project combines a terminal interface imitating a chatbot as well as a real-time chart of your spending, creating an attractive yet simple way to approach budgeting and financial awareness.

Features

- Monthly budget limit: view and compare

- Add and view expenses in multiple categories and subcategories

- Notifications about your budget purchase

- An interactive, user-friendly way of viewing expenses on a pie chart

- Browser-based system with PyScript

- Clean, responsive design layout developed with HTML. /CSS

- Lightweight.

---

Project Overview

The Expense Tracker Chatbot allows users to manage personal or student finances easily.  
Users can enter expenses via a chatbot-like interface, check their spending status, and view category distributions visually.  
It was developed as a **Python mini project** under the guidance of **Prof. Dawa Lepcha** at **Symbiosis Artificial Intelligence Institute**.

Installation & Usage

Prerequisites
- A modern browser (Chrome, Edge, Firefox, or Safari)
- Internet connection (for loading PyScript and Chart.js)

Steps to Run
1. Download or clone this repository.  
2. Open `index.html` in your web browser.  
3. Type anything to begin interacting with the chatbot.  
4. Enter your monthly budget.  
5. Use menu options to add/view expenses or check your budget status.  

---

System Workflow

1. Set Monthly Budget** — Define your total spending limit.  
2. Add Expenses** — Input expense name and amount.  
3. View All Expenses** — Display a detailed expense list.  
4. Budget Status** — Compare total spending vs. remaining balance.  
5. Expense Visualization** — View distribution with a pie chart.

---

Methodology

The project follows a structured development approach:

- **Design & Development:** Created using Python for logic and HTML/CSS for UI.  
- **Data Handling:** Expenses stored in JSON-like structures for simplicity.  
- **Visualization:** Real-time pie charts using Chart.js for financial insights.  
- **Integration:** PyScript bridges Python execution directly within HTML for smooth front-end and back-end communication.
